## 🧠 About This Project

> *A complete Machine Learning pipeline predicting heart disease risk — designed as a robust resource for understanding tree-based algorithms and optimization.*

This project demonstrates the Data Science lifecycle using clinical heart disease data. It emphasizes clean code, robust data pipelines, and clear pedagogical explanations for handling mixed data types, preventing data leakage, and optimizing complex machine learning algorithms to achieve maximum predictive accuracy.

### 🎯 Learning Outcomes
By exploring this project, you will master:
* **Pipeline Architecture**: Building reproducible `Scikit-Learn` pipelines.
* **Feature Engineering**: Handling clinical data using scalers for continuous metrics and encoders for categorical signals.
* **Hyperparameter Tuning**: Optimizing tree-based algorithms (`XGBoost Classifier`) using `GridSearchCV` to prevent overfitting and find the absolute best predictive parameters.
